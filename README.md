# 🐯 Idol_Mixes 🐯

A modern, searchable database for Idol Calls and Mixes.

## 🤖 The Vibe-Coded
This project is **vibe coded**. A **0%** vibe coded one is on the works

## 📚 The Data Source
The foundational logic and call structures used in this database are from this guide:
👉 [Idol Mixes on Scribd](https://www.scribd.com/document/618992902/Idol-Mixes)

If you find a chant here, chances are it was painstakingly transcribed and shared in that document.

## 🛠️ Tech Stack
* **React** - Powering the UI and the search logic.
* **Tailwind CSS** - For that fast, responsive, and sleek "dark mode live house" aesthetic.
* **Vite** - For the lightning-fast dev experience.

## 🚀 Getting Started
To run this locally:

1.  Clone the repo:
    ```bash
    git clone [https://github.com/your-username/your-repo-name.git](https://github.com/your-username/your-repo-name.git)
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Fire it up:
    ```bash
    npm run dev
    ```
